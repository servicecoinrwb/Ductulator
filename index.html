User Guide & Scenarios

Quick Start

  1. Select Residential (Manual D) or Commercial (Manual N) mode.
  2. Drag the slider or type to set your Air Volume (CFM).
  3. Set Friction Rate (Standard: 0.1 for Supply, 0.08 for Return).
  4. Read the Recommended Round Diameter in the blue card.
  5. Use the Rectangular Sizing section to convert round pipe to duct dimensions (e.g., enter 8" height to find width).
1

Check Existing Return Air

Scenario: You are installing a new furnace. Is the old return drop big enough?

  • Step 1: Determine the new furnace's required CFM (e.g., 3-Ton = 1200 CFM). Input this into the slider.
  • Step 2: Set Friction Rate to 0.08 (Standard for Returns).
  • Step 3: Note the Round Size results.
  • Step 4: In Rectangular Sizing, enter the measurement of one side of the existing drop (e.g., 24").
  • Result: If the calculated second side is larger than your actual drop depth, the existing drop is too small and must be replaced to prevent motor failure.
2

Sizing New Supply Trunk

Scenario: Building a main trunk line for a 1000 CFM system with limited headroom.

  • Step 1: Set Volume to 1000 CFM.
  • Step 2: Set Friction Rate to 0.1 (Standard for Supply).
  • Step 3: Check the Velocity Gauge. If it's green (Optimal), proceed. If red, lower friction or increase size.
  • Step 4: Measure max available height (e.g., 8"). Enter 8 into "Side 1".
  • Result: The "Side 2" box tells you the required width (e.g., 22"). Build a 22" x 8" trunk.

Pro Tip: Calculating Correct Friction Rate

Don't guess! Calculate the specific friction rate for your layout using the "Longest Run" method (Manual D).

1. Find Total Effective Length (TEL)

Measure the path to the farthest supply register and farthest return grill. Add them together.

Add Fitting Equivalent Lengths: Don't forget to add equivalent feet for every elbow, takeoff, and boot (e.g., 1 elbow ≈ 30ft).

2. Determine Available Static (ASP)

Start with Blower capability (e.g., 0.50" WC).

Subtract losses for: A/C Coil, Filter, Dampers, and Grills.

0.50 (Blower)
- 0.20 (Coil)
- 0.10 (Filter)
= 0.20" ASP
3. The Formula

Calculate the Friction Rate (FR) to enter in the slider above:

FR = (ASP × 100) / TEL

Example: (0.20 × 100) / 400ft = 0.05 Friction Rate.
